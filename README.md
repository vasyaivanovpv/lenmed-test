## Реализовать приложение на React GraphQL:

Клиент запрашивает полный набор данных о матчах и подписывается на обновление данных через websocket.
Выводится таблица с данными о матчах.
При получении сообщений об изменениях обновляются соответствующие данные о матчах в интерфейсе.

### `yarn start`

### `yarn build`
